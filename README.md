# 绘本记录 H5

零依赖的本地亲子绘本阅读记录 H5。沿用 `明信片H5/` 的极简风格，双击 `index.html` 即可使用。

## 使用

1. 双击 `index.html` 用浏览器打开（推荐 Chrome / Safari）
2. 第一次进入，点右下角「+」开始记录
3. 拍照或从相册选择封面（自动压缩）
4. 填书名（必填）/ 作者 / 阅读日期 / 笔记
5. 可标记 ❤️
6. 保存后回到书架

## 功能

- 📚 **书架**：2 列封面墙，按月份 / ❤️ / 排序筛选
- 📊 **统计**：累计、按月条形图、365 天阅读热力、最近 5 本
- ⚙ **设置**：JSON 备份导出 / 导入 / 清空 / AI 识别（占位）
- 🖼 **海报**：单本书详情页可生成 1080×1440 海报图分享

## 文件

- `index.html` — 单页结构
- `styles.css` — 全部样式（米白 + 暖橘点缀）
- `app.js` — 全部业务（IndexedDB / 图片压缩 / 海报 / 筛选）

## 数据存储

- 浏览器 IndexedDB（数据库名：`parenting_books`）
- 数据**仅本地**，不联网
- 换设备请在「设置」导出 JSON 后再导入

## 浏览器兼容

- ✅ 桌面 Chrome / Safari / Edge
- ✅ iPhone Safari（含拍照）
- ✅ Android Chrome（含拍照）

## 关于 AI 识别

设置页里的「AI 识别」按钮为占位（点击会提示"开发中"）。
未来版本可对接任意多模态大模型（豆包 / GPT-4o 等）。
